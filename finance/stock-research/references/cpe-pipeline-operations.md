# CPE Research Pipeline — Operational Guide

How to run the CPE research agent, interpret results, and run evals. For methodology, see adversarial-research-framework.md.

## Running a Report

```bash
cd ~/projects/avgo
bash run.sh AVGO   # or any ticker
```

`run.sh` handles venv activation, PYTHONPATH, .env setup, and research-cli check.

### Workspace Reuse Behavior (important)
The pipeline uses `workspace/{TICKER}_{DATE}/` as its working directory. If today's workspace already exists with prior output:
- **Phase 1 GATHER:** Skips if `raw/` has >=15 files
- **Phase 2 ANALYZE:** Skips if `analysis/` has >=5 lens files
- **Phase 3.5 BEAR CHALLENGE:** Always re-runs (independent adversarial stress-test)
- **Phase 3 SYNTHESIZE:** Skips if `memo.md` exists
- **Phase 4 EVAL:** Skips if `eval/quality_report.md` exists

**To get a fully clean run:** delete or rename the existing workspace directory first. Otherwise you'll get a partial re-run that only regenerates missing phases.

### Tool Name Auto-Repair
The agent model frequently emits `mcp_read_file`, `mcp_write_file`, `mcp_terminal`, `mcp_skill_view`, etc. Hermes auto-repairs these (logs show `Auto-repaired tool name`). This is normal and not a bug. `Glob` is NOT a valid tool and triggers self-correction (1/3 retries).

### Typical Runtime
- Full run (all 4 phases): ~4-5 minutes
- Partial run (phases 1-2 cached): ~2-3 minutes (bear challenge only)

### Layer 6: LLM-as-Judge (requires API, ~1-2 min)

The judge calls the Anthropic API directly. On this machine, API access goes through the billing proxy at `localhost:18801`.

**Critical setup -- proxy routing:**
```bash
export ANTHROPIC_API_KEY="dummy"           # any non-empty string works
export ANTHROPIC_BASE_URL="http://localhost:18801"  # billing proxy
```

The Anthropic Python SDK auto-reads `ANTHROPIC_BASE_URL` even when `api_key` is passed explicitly to the constructor.

**Model name mismatch:** The judge hardcodes `claude-opus-4-7-20250715` (dated model ID) as `DEFAULT_MODEL`. The proxy only serves alias names (`claude-opus-4-7`, no date suffix). You must override the model at call time:

```python
from evals.layer6_memo_quality.judge import score_memo
# Pass model= explicitly to avoid the 404
scores = score_memo(memo, source_index, model='claude-opus-4-7')
```

The `run.py` CLI (`python -m evals.layer6_memo_quality.run --ticker AVGO`) does NOT accept a `--model` flag, so you can't use it directly through the proxy without patching. Use the Python API above instead.

**Full Layer 6 invocation:**
```bash
cd ~/projects/avgo
source ~/.hermes/hermes-agent/venv/bin/activate
export PYTHONPATH="$HOME/.hermes/hermes-agent:$PWD"
export ANTHROPIC_API_KEY="dummy"
export ANTHROPIC_BASE_URL="http://localhost:18801"

python -c "
from evals.conftest import find_latest_workspace
from evals.layer6_memo_quality.judge import score_memo

ws = find_latest_workspace('AVGO')
memo = (ws / 'memo.md').read_text()
si_path = ws / 'appendix' / 'G_source_index.md'
si = si_path.read_text() if si_path.exists() else ''
scores = score_memo(memo, si, model='claude-opus-4-7')

for dim, data in scores.items():
    if dim in ('composite', 'trials_completed', 'error'):
        continue
    print(f'  {dim}: {data[\"score\"]}/5')
print(f'  COMPOSITE: {scores[\"composite\"]:.2f}/5')
"
```

### Eval Dimensions (7, scored 1-5, 3 trials median)
| Dimension | Target | Notes |
|-----------|--------|-------|
| conviction_clarity | >=4 | Verdict + justified score |
| first_principles_reasoning | >=4 | Causal chains from data |
| directional_insight | >=3 | Non-obvious thesis |
| source_quality | >=4 | Primary sources, cited |
| information_density | >=3 | Every sentence earns its place |
| source_diversity_hierarchy | >=3 | 4+ source types |
| industry_dynamics | >=3 | Supply/demand + competition evolution |
| **composite** | **>=3.5** | Production quality threshold |

## Environment Notes

- **hermes_home/.env:** The pipeline's own `.env` is generated by `scripts/setup_env.sh` from `~/.hermes/.env`. The key name is `ANTHROPIC_TOKEN` (not `ANTHROPIC_API_KEY`) -- this is the billing proxy token.
- **research CLI:** Must be installed via `cd ~/projects/research-cli && uv tool install -e .`. Verify with `which research`.
- **GitHub:** Org is `cpe-research`, PAT is `CPE_GITHUB_TOKEN`, SSH alias is `github-charles`.
