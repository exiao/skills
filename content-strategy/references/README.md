# Meta Ads Creative - References

This folder contains reference materials for the meta-ads-creative skill.

## Reference Files

| File | Purpose | When to Use |
|------|---------|-------------|
| `6-elements-framework.md` | Deep dive into Media, Primary Text, Headline, Description, CTA, Page | When optimizing any element of an ad |
| `ad-formats-library.md` | All ad format types with templates and examples | When selecting format for a new ad |
| `copywriting-formulas.md` | PAS, AIDA, hooks, headlines, awareness levels | When writing ad copy |
| `creative-research-methods.md` | How to find trending formats and build swipe file | Weekly research routine |

## Usage Order

**When creating new ads:**
1. Start with `creative-research-methods.md` - What's working right now?
2. Use `ad-formats-library.md` - Which format fits this message?
3. Apply `copywriting-formulas.md` - How to write the copy?
4. Check `6-elements-framework.md` - Is each element optimized?

**When optimizing existing ads:**
1. Review `6-elements-framework.md` - Which element is underperforming?
2. Reference `copywriting-formulas.md` - Can we improve the copy?
3. Consider `ad-formats-library.md` - Should we test a different format?
