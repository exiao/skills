## Using This List

When recommending ideas:

1. Ask about product, audience, and current stage if not clear
2. Suggest 3-5 most relevant ideas based on context
3. Provide implementation steps for chosen ideas
4. Consider resources (time, budget, team size)

For each recommendation, provide:
- **Idea name**: One-line description
- **Why it fits**: Connection to their situation
- **How to start**: First 2-3 implementation steps
- **Expected outcome**: What success looks like
- **Resources needed**: Time, budget, skills required
