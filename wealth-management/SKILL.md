# Wealth Management

Client-facing wealth management workflows. Use when preparing for client meetings, building financial plans, generating investment proposals, rebalancing portfolios, or identifying tax-loss harvesting opportunities.

## Sub-Skills

| Task | When to use |
|------|-------------|
| **client-review** | Before quarterly/annual client meetings — performance summary, talking points, action items |
| **client-report** | Generate professional client-facing performance reports (PDF/Word) |
| **financial-plan** | Build or update comprehensive financial plans — retirement, education, estate, cash flow |
| **investment-proposal** | Create proposals for prospective clients — allocation, expected outcomes, fee structure |
| **portfolio-rebalance** | Analyze drift and generate tax-aware rebalancing trades |
| **tax-loss-harvesting** | Identify TLH opportunities, suggest replacements, track wash sale windows |
