# Output Directory Structure

All atomized content goes to:

```
~/marketing/atomized/YYYY-MM-DD/
├── atoms.md                    # Step 0 extraction (the raw atoms)
├── x-thread.md                 # X/Twitter thread
├── x-standalone-tweets.md      # Standalone topic tweets
├── linkedin-post.md            # LinkedIn post + first comment
├── tiktok-script.md            # TikTok video script
├── ig-carousel.md              # Instagram carousel slide-by-slide
├── ig-reel-script.md           # Instagram Reels script
├── yt-short-script.md          # YouTube Shorts script
├── threads-post.md             # Threads post(s)
└── substack-excerpt.md         # Substack newsletter excerpt
```

Create the directory if it doesn't exist. Use today's date.

If atomizing multiple articles on the same day, append a slug: `YYYY-MM-DD-slug/`.
