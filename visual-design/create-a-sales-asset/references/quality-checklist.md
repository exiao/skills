# Quality Checklist

### Content
- [ ] Company name spelled correctly throughout
- [ ] Leadership names are current
- [ ] Pain points accurately reflect input
- [ ] No placeholder text remaining
- [ ] Proof points are accurate

### Visual
- [ ] Brand colors applied correctly
- [ ] All text readable (contrast)
- [ ] Animations smooth, not distracting
- [ ] Responsive (if interactive page)

### Functional
- [ ] All tabs/sections load
- [ ] Interactive elements work
- [ ] Workflow steps animate properly
- [ ] Navigation is intuitive
- [ ] CTA is clear

### Professional
- [ ] Tone matches audience
- [ ] Appropriate detail level
- [ ] No typos
- [ ] Feels tailored, not templated
