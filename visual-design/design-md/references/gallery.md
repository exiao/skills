# VoltAgent DESIGN.md Gallery

68 ready-to-use DESIGN.md templates from [VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md), extracted from the public websites of well-known brands.

**Install any of these:**
```bash
npx getdesign@latest add <brand>   # e.g. npx getdesign@latest add vercel
```

If `DESIGN.md` already exists at project root, the CLI saves to `./<brand>/DESIGN.md` instead. To activate: `cp <brand>/DESIGN.md ./DESIGN.md`.

**List all available templates:**
```bash
npx getdesign@latest list
```

---

## How to pick

Ask the user which *category* feels closest to the aesthetic they want, then show only brands in that category. Faster than dumping the whole list.

Quick mental map:
- **Warm editorial / premium human** → claude, airbnb, ferrari
- **Cold precise / developer** → vercel, linear.app, raycast, warp
- **Dark cinematic** → tesla, bugatti, runwayml, elevenlabs
- **Playful colorful** → figma, airtable, lovable, clay
- **Data-dense SaaS** → notion, posthog, supabase, clickhouse
- **Monochrome minimalist** → apple, x.ai, hashicorp, bugatti
- **Bold accent on monochrome** → binance (yellow), coinbase (blue), stripe (purple), sentry (orange)
