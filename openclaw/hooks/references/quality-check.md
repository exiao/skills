# Quality Check

Before publishing any headline, it must pass ALL of these:
1. **Concert test** (Concert Theory): If you said this on stage to 25,000 people, would they lean in — or pull out their phones? If "phones," it's dead. See SKILL.md → Concert Theory.
2. **Bar test**: Would you say this to a friend at a bar? If it sounds weird spoken aloud, rewrite.
3. **Group chat test** (Signulll): Would someone share this? To virtue signal, say something they can't say themselves, or show off their taste?
4. **Second sentence test**: Does it make you NEED to read the next line?
5. **Specificity test**: Are there concrete numbers, names, or details? ("363 tests" not "many tests")
6. **Morning test**: Write it, sleep on it. If you still like it tomorrow, it's good. 90% you'll hate it and rewrite.
7. **One topic test**: Does it promise exactly one thing? Not two, not a vague umbrella.
8. **Organic test**: Does it sound like content or like an ad? The second it sounds like an ad, it's dead.
9. **Tension-not-context test**: Does the first line open with tension, or with setup? If it's setup, cut it — context is owed *after* attention, not before.
